If the outer page (here) and the framed page (within the rectangle) are on different domains, they aren't normally permitted to communicate directly. In this demo, however, the rectangle above is an instance of <basic-seamless-iframe>, and the framed page hosts an instance of the companion component <basic-framed-content>. These two components cooperate across the frame boundary. This lets the outer page see that the content of the framed page is: "". This also lets the frame automatically size itself to the content of the framed page.
